# 行为准则

## 我们的承诺

为了建立一个开放和欢迎的环境,我们每一个贡献者和维护者都承诺:

- 参与我们社区的每个人都都能够获得没有骚扰的体验,无论其年龄、身体特征、残疾、种族、性别认同与表现、经验水平、教育程度、社会经济地位、国籍、个人形象、种族、宗教或性认同与倾向
- 我们承诺以强化开放、欢迎、多样化、包容和健康社区的方式采取行动

## 我们的标准

有益于我们社区环境的行为示例包括:

- 使用欢迎和包容的语言
- 尊重不同的观点和经验
- 接受建设性的批评
- 专注于对社区最有利的事物
- 对其他社区成员表示同情

不可接受的行为示例包括:

- 带有性含义的语言或意象和不欢迎的性关注或进展
- 羞辱性言论、侮辱性/贬低性评论以及人身或政治攻击
- 公共或私人骚扰
- 在未明确许可的情况下发布他人的私人信息,如物理地址或电子邮件地址
- 其他在专业环境中合理认为是不适当的行为

## 执行

项目维护者负责阐明可接受行为的标准,并应对任何不可接受行为采取适当、公平的纠正措施。

项目维护者有权利和责任删除、编辑或拒绝与本行为准则不相符的评论、提交、代码、Wiki 编辑、问题和其他贡献,以及暂时或永久禁止任何贡献者进行他们认为不适当、威胁、冒犯或有害的其他行为。

## 范围

本行为准则适用于所有项目空间,也适用于个人在公开代表该项目或其社区时的行为。公开代表项目的示例包括使用官方项目电子邮件地址、通过官方社交媒体帐户发布或作为活动中指定的代表。项目维护者可以进一步定义和澄清项目代表。

## 执行

辱骂性、骚扰性或其他不可接受的行为可以通过与项目团队联系来报告。所有投诉将被审查和调查,并将导致被认为必要和适当的回应。项目团队有义务为事件报告者保持保密。具体执行政策的进一步详情可能被另外发布。

不善意地遵循或执行行为准则的项目维护者可能会面临由项目领导确定的临时或永久影响。
